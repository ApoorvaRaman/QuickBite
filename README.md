# QuickBite 🍔

### Tagline
Crave it. Click it. QuickBite delivers.

### Description
QuickBite is a frontend-only online food ordering system built for a DBMS group project. It simulates real-world food delivery platforms.

### Features
- User Login UI
- Restaurant browsing
- Add to Cart system
- Payment simulation
- Order tracking UI

### Tech Stack
- HTML
- CSS
- JavaScript

### How to Run
1. Download repo
2. Open login.html
3. Navigate through system
